บทที่ 8 : PHP Abstract Classes

คลาสนามธรรม (Abstract Classes)

แม่พิมพ์แบบร่างที่บังคับให้คลาสลูกต้องนำไปเขียนรายละเอียดการทำงานต่อ (Implement) ด้วยตัวเอง เพื่อสร้างมาตรฐานเดียวกัน

Abstract Class คืออะไร?

คลาสนามธรรม (Abstract Class) คือคลาสที่มีคำว่า abstract นำหน้า โดยมีจุดประสงค์หลักเพื่อเป็นแบบร่างให้คลาสลูกสืบทอดไปใช้งาน มีกฎสำคัญ 2 ข้อคือ:

  • ❌ ไม่สามารถสร้าง Object ตรงๆ ได้: คำสั่ง new AbstractClass() จะเกิด Error เสมอ ต้องสร้างผ่านคลาสลูกที่ extends ไปเท่านั้น
  • ⚠️ Abstract Method บังคับเขียนต่อ: หากในคลาสมีเมธอดที่ขึ้นต้นด้วย abstract (จะไม่มีปีกกาไม่มีเนื้อหาในเมธอด) คลาสลูกที่สืบทอดไป "บังคับต้องเขียนเนื้อหาของเมธอดนั้นให้ครบถ้วน"

ตัวอย่างการใช้งาน Abstract Class และ Method

PHP 8+
abstract_classes.php 
<?php
// 1. สร้าง Abstract Class (คลาสนามธรรมแม่แบบ)
abstract class Car {
  public $name;
  public function __construct($name) {
    $this->name = $name;
  }

  // Abstract Method (มีแค่ชื่อ ห้ามมีปีกกาเนื้อหา บังคับคลาสลูกต้องเขียนเอง!)
  abstract public function intro(): string;
}

// 2. คลาสลูก Audi สืบทอดไป และ "ต้อง" เขียนเมธอด intro()
class Audi extends Car {
  public function intro(): string {
    return "🚗 คุณภาพเยอรมัน! ฉันคือ {$this->name}";
  }
}

// 3. คลาสลูก Volvo
class Volvo extends Car {
  public function intro(): string {
    return "🛡️ ปลอดภัยอันดับหนึ่ง! ฉันคือ {$this->name}";
  }
}

// $car = new Car("Test"); ❌ Error! ไม่สามารถสร้าง Object จาก Abstract Class ได้
$audi = new Audi("Audi R8");
echo $audi->intro(); // ผลลัพธ์: 🚗 คุณภาพเยอรมัน! ฉันคือ Audi R8
echo "<br>";

$volvo = new Volvo("Volvo XC90");
echo $volvo->intro(); // ผลลัพธ์: 🛡️ ปลอดภัยอันดับหนึ่ง! ฉันคือ Volvo XC90
?>
