บทที่ 14 : PHP Iterables

การวนลูปข้อมูล (PHP Iterables)

ประเภทข้อมูลจำลอง (Pseudo-type) ที่ระบุว่าค่าข้อมูลนั้นๆ สามารถนำมาวนรอบด้วยคำสั่ง foreach() ได้อย่างถูกต้อง

Iterable คืออะไร?

ใน PHP 7.1 ขึ้นไป มีการเพิ่มประเภทข้อมูลจำลอง (Pseudo-type) ที่เรียกว่า iterable ซึ่งหมายถึง ข้อมูลใดๆ ก็ตามที่สามารถนำมาวนลูปด้วยคำสั่ง foreach() ได้

ข้อมูลที่เป็น iterable ได้แก่ Array ทั่วไป หรือ Object ที่ใช้งานอินเทอร์เฟซ Traversable / Iterator เรามักใช้คำว่า iterable ในการกำหนด Type Hint ในพารามิเตอร์หรือค่าคืนกลับของฟังก์ชัน เพื่อให้โค้ดรัดกุมและตรวจสอบข้อผิดพลาดได้ง่าย

ตัวอย่างการใช้ Type Hint เป็น iterable

PHP 8+
iterables.php 
<?php
// 1. ฟังก์ชันที่รับพารามิเตอร์เป็นประเภท iterable
function printIterable(iterable $myIterable) {
  foreach ($myIterable as $item) {
    echo "👉 รายการ: {$item}<br>";
  }
}

// 2. ฟังก์ชันที่กำหนดค่าคืนกลับ (Return Type) เป็น iterable
function getIterable(): iterable {
  return ["Apple", "Banana", "Cherry"];
}

// ทดสอบเรียกใช้งาน
$myArray = getIterable();
printIterable($myArray);

/* 
ผลลัพธ์:
👉 รายการ: Apple
👉 รายการ: Banana
👉 รายการ: Cherry
*/
?>
