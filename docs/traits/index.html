บทที่ 10 : PHP Traits

เทรตและยืมโค้ดร่วมกัน (PHP Traits)

ตัวช่วยแก้ข้อจำกัด Single Inheritance ทำให้คุณสามารถนำชุดเมธอดไปใช้ร่วมกันในหลายๆ คลาสได้อย่างอิสระโดยไม่ต้องสืบทอด

Trait คืออะไร? ทำไมต้องใช้?

ในภาษา PHP คลาส 1 คลาสสามารถสืบทอด (extends) จากคลาสแม่ได้เพียงคลาสเดียวเท่านั้น (Single Inheritance) ซึ่งหากเรามีชุดฟังก์ชันบางอย่าง (เช่น ระบบตัดคำ, ระบบบันทึก Log) ที่อยากให้คลาสที่ไม่เกี่ยวข้องกันเลยนำไปใช้ร่วมกัน จะทำอย่างไร?

คำตอบคือการใช้ Trait! เราสร้าง Trait ด้วยคีย์เวิร์ด trait และนำเข้าไปใช้ในคลาสใดก็ได้ด้วยคีย์เวิร์ด use แถมยังสามารถ use หลายๆ Trait พร้อมกันได้อีกด้วย!

ตัวอย่างการใช้งาน trait และ use

PHP 8+
traits.php 
<?php
// 1. สร้าง Trait สำหรับบันทึกข้อความ Log
trait LoggerTrait {
  public function log($msg) {
    echo "📝 [LOG]: {$msg}<br>";
  }
}

// 2. สร้าง Trait สำหรับแจ้งเตือน
trait NotifierTrait {
  public function notify() {
    echo "🔔 [ALERT]: มีการแจ้งเตือนใหม่ในระบบ!<br>";
  }
}

// 3. คลาส User ดึงทั้ง 2 Trait มาใช้งานร่วมกันด้วยคำสั่ง use
class User {
  use LoggerTrait, NotifierTrait;
}

// 4. คลาส Product ดึงมาใช้แค่ LoggerTrait
class Product {
  use LoggerTrait;
}

$user = new User();
$user->log("ผู้ใช้เข้าสู่ระบบ");
$user->notify();
echo "<hr>";

$product = new Product();
$product->log("เพิ่มสินค้าใหม่ลงตะกร้า");
?>
