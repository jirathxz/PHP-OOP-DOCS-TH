บทที่ 6 : PHP Inheritance

การสืบทอดคุณสมบัติ (PHP Inheritance)

การสร้างคลาสลูกที่รับคุณสมบัติและพฤติกรรมจากคลาสแม่ด้วยคำสั่ง extends การ Overriding เมธอด และการใช้คำสั่ง final

Inheritance คืออะไร?

Inheritance คือการที่ คลาสลูก (Child Class / Subclass) สามารถรับเอา Properties และ Methods ที่เป็น public และ protected จาก คลาสแม่ (Parent Class / Superclass) มาใช้งานได้ทันที

ใน PHP เราใช้คีย์เวิร์ด extends ในการสืบทอด (ข้อจำกัดคือ PHP รองรับ Single Inheritance นั่นคือคลาสลูกสามารถสืบทอดจากคลาสแม่ได้เพียงคลาสเดียวเท่านั้น)

ตัวอย่างการใช้ extends และ Overriding

PHP 8+
inheritance.php 
<?php
// 1. คลาสแม่ (Parent Class)
class Fruit {
  public $name;
  public $color;

  public function __construct($name, $color) {
    $this->name = $name;
    $this->color = $color;
  }

  public function intro() {
    echo "ฉันคือผลไม้ชื่อ {$this->name} สี {$this->color}<br>";
  }
}

// 2. คลาสลูก (Child Class) สืบทอดจาก Fruit ด้วยคำสั่ง extends
class Strawberry extends Fruit {
  public function message() {
    echo "🍓 สตรอว์เบอร์รีเป็นผลไม้หรือเบอร์รีกันนะ?<br>";
  }

  // Method Overriding: เขียนทับเมธอด intro() ของคลาสแม่ให้ทำงานแบบใหม่!
  public function intro() {
    echo "✨ [Overridden] ฉันคือสตรอว์เบอร์รีรสเปรี้ยวอมหวาน สี {$this->color}<br>";
  }
}

$strawberry = new Strawberry("Strawberry", "Red");
$strawberry->message(); // เรียกเมธอดของตัวเอง
$strawberry->intro();   // เรียกเมธอดที่ถูกเขียนทับ (Overriding)
?>

คีย์เวิร์ด final (ห้ามสืบทอดหรือเขียนทับ)

หากคุณไม่ต้องการให้ใครนำคลาสของคุณไปสืบทอดต่อ หรือไม่ต้องการให้เมธอดใดถูกเขียนทับ (Override) คุณสามารถเติมคำว่า final ไว้ด้านหน้าได้:

final class SecretClass {} // ❌ คลาสนี้ห้ามใช้ extends ต่อเด็ดขาด
final public function secureMethod() {} // ❌ เมธอดนี้ห้ามทำ Overriding เด็ดขาด
