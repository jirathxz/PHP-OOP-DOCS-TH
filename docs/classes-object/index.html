บทที่ 2 : PHP Classes / Objects

คลาสและออบเจกต์ (Classes / Objects)

เจาะลึกวิธีการสร้าง Class การประกาศ Object ด้วยคำสั่ง new การใช้ตัวแปร $this และตัวช่วยตรวจสอบประเภท instanceof

Class กับ Object ทำงานร่วมกันอย่างไร?

ในภาษา PHP เราประกาศคลาสโดยใช้คีย์เวิร์ด class ตามด้วยชื่อคลาส โดยภายในคลาสจะบรรจุตัวแปร (เรียกว่า Properties) และฟังก์ชัน (เรียกว่า Methods)

เมื่อเราต้องการนำคลาสนั้นมาสร้างเป็นตัวตนจริงในหน่วยความจำ เราจะใช้คีย์เวิร์ด new เพื่อสร้าง Object (หรือ Instance) ซึ่งแต่ละออบเจกต์จะทำงานอิสระจากกันอย่างสิ้นเชิง

คีย์เวิร์ด $this คืออะไร?

คีย์เวิร์ด $this เป็นตัวแทนที่ใช้อ้างอิงถึง "ออบเจกต์ปัจจุบัน (Current Object)" ที่กำลังเรียกใช้งานเมธอดนั้นอยู่ เวลาเราต้องการเข้าถึง Property หรือ Method ภายในคลาสเดียวกัน เราจะต้องใช้ $this->property_name เสมอ (ข้อสังเกต: หลังเครื่องหมาย -> จะไม่มีเครื่องหมาย $ หน้าชื่อตัวแปร)

ตัวอย่างการสร้าง Class และ Object ใน PHP

PHP 8+
classes_objects.php 
<?php
class Car {
  // ประกาศ Properties
  public $brand;
  public $color;

  // ประกาศ Methods โดยใช้ $this เพื่ออ้างอิงออบเจกต์ปัจจุบัน
  public function set_brand($brand) {
    $this->brand = $brand;
  }
  
  public function get_brand() {
    return $this->brand;
  }
}

// สร้าง Object จากคลาส Car ด้วยคำสั่ง new
$myCar1 = new Car();
$myCar2 = new Car();

// กำหนดค่าให้แต่ละออบเจกต์ (อิสระต่อกัน)
$myCar1->set_brand("Toyota");
$myCar2->set_brand("Honda");

echo $myCar1->get_brand(); // ผลลัพธ์: Toyota
echo "<br>";
echo $myCar2->get_brand(); // ผลลัพธ์: Honda

// การตรวจสอบประเภทด้วย instanceof
var_dump($myCar1 instanceof Car); // ผลลัพธ์: bool(true)
?>

คีย์เวิร์ด instanceof

คุณสามารถใช้คำสั่ง instanceof ในการตรวจสอบว่าตัวแปร Object นั้นๆ ถูกสร้างมาจากคลาสที่ระบุหรือไม่ โดยจะคืนค่ากลับมาเป็น boolean (true หรือ false)
