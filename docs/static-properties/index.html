บทที่ 12 : PHP Static Properties

คุณสมบัติสถิต (Static Properties)

ตัวแปรสถิตที่ใช้ร่วมกันในทุก Instance สามารถเรียกใช้ได้ทันทีผ่าน ClassName::$property โดยไม่ต้องสร้าง Object

Static Property คืออะไร?

เช่นเดียวกับ Static Method คุณสามารถประกาศตัวแปรให้เป็น static ได้ ซึ่งทำให้ตัวแปรนั้นถูกจัดเก็บเป็นส่วนกลาง (Shared Value) สำหรับคลาสนั้นๆ ไม่ได้ผูกติดกับ Object ใด Object หนึ่ง

ข้อสังเกตสำคัญ: เวลาเรียกใช้ Static Property ผ่านเครื่องหมาย :: จะต้องมีเครื่องหมาย $ นำหน้าชื่อตัวแปรเสมอ (เช่น ClassName::$value) ซึ่งต่างจาก Constants

ตัวอย่างการประกาศและใช้งาน static property

PHP 8+
static_properties.php 
<?php
class Pi {
  // ประกาศ Static Property
  public static $value = 3.14159;

  public function staticValue() {
    // เข้าถึงจากภายในคลาสใช้ self::$value
    return self::$value;
  }
}

// 1. เรียกใช้งานจากภายนอกโดยไม่ต้องใช้ new (ต้องมีเครื่องหมาย $ หน้าชื่อตัวแปร)
echo "ค่า Pi คือ: " . Pi::$value;
echo "<br>";

// 2. เรียกผ่าน Object
$pi = new Pi();
echo "ค่า Pi จากออบเจกต์: " . $pi->staticValue();
?>
