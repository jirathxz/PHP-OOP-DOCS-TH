บทที่ 11 : PHP Static Methods

เมธอดสถิต (Static Methods)

เมธอดที่สามารถเรียกใช้งานได้ทันทีโดยไม่ต้องสร้าง Object ผ่านคำสั่ง ClassName::methodName()

Static Method คืออะไร?

ตามปกติแล้ว เวลาเราต้องการเรียกใช้เมธอดในคลาส เราจะต้องสร้าง Object ขึ้นมาก่อนด้วยคำสั่ง new แต่สำหรับ Static Method (เมธอดที่เติมคำว่า static ไว้ด้านหน้า) คุณสามารถเรียกใช้งานจากชื่อคลาสได้โดยตรง!

เหมาะสำหรับฟังก์ชันอรรถประโยชน์ (Utility/Helper Functions) เช่น ฟังก์ชันแปลงวันที่, ฟังก์ชันคำนวณคณิตศาสตร์ หรือการเช็กความถูกต้องของข้อมูล

ตัวอย่างการประกาศและเรียกใช้ static method

PHP 8+
static_methods.php 
<?php
class Greeting {
  // ประกาศ Static Method ด้วยคีย์เวิร์ด static
  public static function welcome() {
    return "🎉 ยินดีต้อนรับสู่โลกของ PHP Static Methods!<br>";
  }

  public function test() {
    // การเรียกใช้ static method จาก "ภายในคลาสเดียวกัน" ให้ใช้ self::
    echo self::welcome();
  }
}

// 1. เรียกใช้งานจากภายนอกโดยไม่ต้องมี new (ชื่อคลาส::ชื่อเมธอด)
echo Greeting::welcome();

// 2. เรียกผ่าน Object ทั่วไป
$obj = new Greeting();
$obj->test();
?>
