บทที่ 4 : PHP Destructor

ตัวทำลายออบเจกต์ (PHP Destructor)

เมธอดพิเศษ __destruct() ที่จะทำงานอัตโนมัติก่อนที่ Object จะถูกทำลายหรือเมื่อจบการทำงานของสคริปต์

Destructor คืออะไร?

Destructor เป็นเมธอดพิเศษคู่ตรงข้ามกับ Constructor โดยใช้ชื่อว่า __destruct() (ขีดล่างสองตัวเช่นกัน)

เมธอดนี้จะถูกเรียกทำงานโดยอัตโนมัติที่ส่วนท้ายสุดของสคริปต์ หรือเมื่อออบเจกต์นั้นถูกทำลายออกจากหน่วยความจำ มักใช้สำหรับงานทำความสะอาด (Cleanup) เช่น การปิดการเชื่อมต่อฐานข้อมูล (Close Database Connection), การบันทึกปูมการทำงาน (Logging) หรือลบไฟล์ชั่วคราว

ตัวอย่างการใช้ __destruct()

PHP 8+
destructor.php 
<?php
class Fruit {
  public $name;
  public $color;

  public function __construct($name, $color) {
    $this->name = $name;
    $this->color = $color;
    echo "✅ สร้างออบเจกต์: {$this->name} สำเร็จ!<br>";
  }

  // เมธอด Destructor ทำงานตอนจบสคริปต์หรือเมื่อออบเจกต์ถูกทำลาย
  public function __destruct() {
    echo "💥 ทำลายออบเจกต์: ผลไม้ {$this->name} (สี {$this->color}) ถูกลบออกจากระบบแล้ว!<br>";
  }
}

$apple = new Fruit("Apple", "Red");
echo "--- กำลังประมวลผลการทำงานอื่นๆ ในสคริปต์ ---<br>";

/* 
ผลลัพธ์ที่ได้บนหน้าจอ:
✅ สร้างออบเจกต์: Apple สำเร็จ!
--- กำลังประมวลผลการทำงานอื่นๆ ในสคริปต์ ---
💥 ทำลายออบเจกต์: ผลไม้ Apple (สี Red) ถูกลบออกจากระบบแล้ว!
*/
?>
