บทที่ 5 : PHP Access Modifiers

ระดับการเข้าถึงข้อมูล (Access Modifiers)

การกำหนดสิทธิ์และความปลอดภัยในการเข้าถึง Properties และ Methods (Public, Protected, Private) เพื่อควบคุมความถูกต้องของข้อมูล (Encapsulation)

Access Modifiers ทั้ง 3 ระดับ

ใน PHP คุณสามารถควบคุมได้ว่าใครบ้างที่สามารถอ่านหรือแก้ไขข้อมูลในคลาสได้ โดยใช้คีย์เวิร์ด 3 ตัวนี้:

1. public (ค่าเริ่มต้น)

สามารถเข้าถึงได้จากทุกที่ ทั้งจากภายในคลาส ภายนอกคลาส หรือคลาสลูกที่สืบทอดไป

2. protected

สามารถเข้าถึงได้เฉพาะภายในคลาสของตัวเอง และคลาสลูกที่สืบทอด (Inheritance) ไปเท่านั้น (ภายนอกห้ามเรียก)

3. private

มีความลับสุดยอด! เข้าถึงได้เฉพาะภายในคลาสของตัวเองเท่านั้น แม้แต่คลาสลูกก็ไม่สามารถเข้าถึงได้

ตัวอย่างการใช้งาน Access Modifiers

PHP 8+
access_modifiers.php 
<?php
class Fruit {
  public $name;       // เข้าถึงได้ทั่วไป
  protected $color;   // เข้าถึงได้ในคลาสนี้และคลาสลูก
  private $weight;    // เข้าถึงได้เฉพาะในคลาสนี้เท่านั้น

  public function set_weight($weight) {
    // เมธอด public สามารถเข้าถึง $weight ที่เป็น private ได้เพราะอยู่ในคลาสเดียวกัน
    $this->weight = $weight;
  }
}

$mango = new Fruit();

$mango->name = 'Mango';    // ✅ ทำงานได้ปกติ (Public)
// $mango->color = 'Yellow';  // ❌ Fatal Error: Cannot access protected property
// $mango->weight = '300g';   // ❌ Fatal Error: Cannot access private property

// วิธีที่ถูกต้องในการตั้งค่า private คือผ่านเมธอด public ที่เตรียมไว้ (Encapsulation)
$mango->set_weight('300g'); // ✅ ทำงานได้ปกติ
?>
