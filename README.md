## 📚 สารบัญเนื้อหาทั้ง 14 หัวข้อ

| บทที่ | หัวข้อเรื่อง (Topic) | คำอธิบายย่อ |
| :---: | :--- | :--- |
| **1** | [PHP What is OOP](https://jirathxz.github.io/OOP-DOCS-TH/docs/what-is-oop/) | ทำความเข้าใจแนวคิด OOP คืออะไร ทำไมต้องใช้ และเปรียบเทียบกับ Procedural |
| **2** | [PHP Classes / Objects](https://jirathxz.github.io/OOP-DOCS-TH/docs/classes-object/) | การสร้าง Class (แม่พิมพ์), การประกาศ Object (new), และการใช้ `$this` |
| **3** | [PHP Constructor](https://jirathxz.github.io/OOP-DOCS-TH/docs/constructor/) | เมธอดพิเศษ `__construct()` ที่ทำงานอัตโนมัติตอนสร้าง Object |
| **4** | [PHP Destructor](https://jirathxz.github.io/OOP-DOCS-TH/docs/destructor/) | เมธอดพิเศษ `__destruct()` ที่ทำงานอัตโนมัติตอนจบการทำงานของสคริปต์ |
| **5** | [PHP Access Modifiers](https://jirathxz.github.io/OOP-DOCS-TH/docs/access-modifiers/) | การกำหนดสิทธิ์เข้าถึงข้อมูล `public`, `protected`, และ `private` |
| **6** | [PHP Inheritance](https://jirathxz.github.io/OOP-DOCS-TH/docs/inheritance/) | การสืบทอดคุณสมบัติด้วย `extends`, การ Overriding เมธอด, และคำสั่ง `final` |
| **7** | [PHP Constants](https://jirathxz.github.io/OOP-DOCS-TH/docs/constants/) | ค่าคงที่ในคลาสด้วย `const` และการเรียกใช้งานผ่าน `self::` และ `::` |
| **8** | [PHP Abstract Classes](https://jirathxz.github.io/OOP-DOCS-TH/docs/abstract-classes/) | คลาสนามธรรมที่เป็นแม่พิมพ์บังคับให้คลาสลูกต้องนำไปเขียนเนื้อหาต่อ |
| **9** | [PHP Interfaces](https://jirathxz.github.io/OOP-DOCS-TH/docs/interfaces/) | การสร้างข้อตกลงและมาตรฐานให้คลาสด้วย `interface` และ `implements` |
| **10** | [PHP Traits](https://jirathxz.github.io/OOP-DOCS-TH/docs/traits/) | ตัวช่วยแก้ข้อจำกัด Single Inheritance ทำให้ใช้ชุดเมธอดร่วมกันในหลายคลาสได้ |
| **11** | [PHP Static Methods](https://jirathxz.github.io/OOP-DOCS-TH/docs/static-methods/) | เมธอดสถิตที่เรียกใช้ได้ทันทีจากชื่อคลาสโดยไม่ต้องสร้าง Object |
| **12** | [PHP Static Properties](https://jirathxz.github.io/OOP-DOCS-TH/docs/static-properties/) | คุณสมบัติสถิตที่ใช้ร่วมกันในทุก Instance ของคลาส |
| **13** | [PHP Namespaces](https://jirathxz.github.io/OOP-DOCS-TH/docs/namespaces/) | การจัดหมวดหมู่คลาส เปรียบเสมือนโฟลเดอร์ ป้องกันปัญหาชื่อคลาสซ้ำกัน |
| **14** | [PHP Iterables](https://jirathxz.github.io/OOP-DOCS-TH/docs/iterables/) | ประเภทข้อมูลที่สามารถนำมาวนลูปด้วยคำสั่ง `foreach()` ได้ |

## 🛠️ เทคโนโลยีและคุณสมบัติเด่นของโปรเจกต์
* **💻 PHP 8+ Syntax:** ตัวอย่างโค้ดทั้งหมดเขียนด้วยไวยากรณ์สมัยใหม่ อ่านง่าย พร้อมคำอธิบายภาษาไทยทีละบรรทัด
* **🎨 Modern Dark Mode UI:** ดีไซน์สไตล์ Glassmorphism เรียบหรู สบายตา ปรับขนาดความสูงการ์ดให้พอดีกับเนื้อหา (Content-Fitted Height)
* **⚡ Dynamic Navigation via JavaScript:** ใช้ระบบ Fetch API ในการโหลดแถบนำทางจาก `nav.html` ทำให้จัดการเมนูง่ายและไม่เขียนโค้ดซ้ำซ้อน
* **📋 Interactive Copy Code:** ทุกกล่องตัวอย่างโค้ดมาพร้อมปุ่ม **Copy** คัดลอกโค้ดไปทดลองรันได้ในคลิกเดียว

## 💻 การติดตั้งและการรันบนเครื่องถิ่น (Local Development)

หากคุณต้องการรันโปรเจกต์นี้บนเซิร์ฟเวอร์จำลองในเครื่อง (เช่น XAMPP, MAMP, Laragon, หรือ Docker):

1. โคลนคลังข้อมูลนี้ลงในโฟลเดอร์เว็บเซิร์ฟเวอร์ของคุณ (เช่น `c:\xampp\htdocs\`):
   ```bash
   git clone https://github.com/jirathxz/OOP-DOCS-TH.git
   ```
2. เปิดโปรแกรมเว็บเบราว์เซอร์และเข้าไปที่:
   ```text
   http://localhost/OOP-DOCS-TH/index.php
   ```
   *(หรือ `http://localhost/OOP-DOCS-TH/index.html` ก็จะเปลี่ยนเส้นทางไปยังหน้าหลักอัตโนมัติ)*
